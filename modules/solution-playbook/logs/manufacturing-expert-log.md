### 3. case-manufacturing-cmf.html（CMF设计）

**专业术语强化**:
- CMF 全称：Color, Material, Finish（色彩、材质、表面处理）
- PBR 纹理：diffuse, roughness, metallic
- 表面处理工艺：阳极氧化、喷涂、电镀、压纹

**痛点描述优化**:
| 原内容 | 优化后 |
|--------|--------|
| 每样$200-500 | 物理打样$200-500/件（喷涂、阳极氧化、涂层） |
| 每张渲染2-4小时 | 材质设置2-4h（Keyshot/VRED材质库+灯光+渲染队列） |
| 白模限制想象 | 白模沟通障碍：决策层难以评估最终CMF效果 |
| 需要UV拆解 | UV拆解瓶颈：任何贴图前需1-2天UV工作 |

**材质展示优化**:
- 从通用名称改为工艺名称
  - 木纹 → 胡桃木纹（木纹贴皮/转印）
  - 金属 → 拉丝阳极氧化铝
  - 塑料红 → 哑光喷涂红
  - 塑料蓝 → 高光注塑蓝（免喷涂注塑）
  - 皮革 → 真皮压纹
  - 黄铜 → 镜面黄铜（电镀抛光）

### 5. case-manufacturing-ideation.html（概念设计）

**专业术语强化**:
- ID (Industrial Design) 工业设计
- Form Factor Validation 形态验证
- Form Model 形态模型
- Ergonomics 人机工程

**痛点描述优化**:
| 原内容 | 优化后 |
|--------|--------|
| 草图快，3D慢 | 草图10分钟，SolidWorks/Rhino建模2-3天 |
| 难传达体积感 | 2D无法验证人机工程（握持感、比例、人机关系） |
| CAD瓶颈 | CAD团队依赖：迭代被排期阻塞1-2周 |
| 迭代成本高 | 每次设计修改 = 重新CAD建模 |

**量化指标优化**:
| 原指标 | 优化后 | 依据 |
|--------|--------|------|
| CAD返工↓80% | 无效CAD工作↓80% | 更精准描述价值 |
| 外部建模$0 | 概念验证成本$0 | 更清晰的价值定位 |
