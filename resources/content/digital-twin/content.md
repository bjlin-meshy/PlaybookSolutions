## 痛点 / Pain Points

### English
- Realistic hair = Print nightmare (strands <0.2mm unprintable)
- Face supports = Surface damage (20%+ rework rate)
- Ultra-fine layer height = Slow (8-14 hours per 10cm portrait)
- $300-800 per likeness sculpt for professional work

### 中文
- 写实发丝 = 打印噩梦（<0.2mm 无法打印）
- 面部支撑 = 表面损伤（返工率 >20%）
- 超细层高 = 漫长等待（10cm 人像 8-14 小时）
- 专业写实雕刻 $300-800/个

---

## 解决方案 / Meshy Solution

### English
- Q-Style = FDM-Safe Geometry (min wall >1.2mm, max overhang <40°)
- Solid Hair Volumes (not strands) - printable on $200 FDM
- 2-5 min per portrait generation
- API batch: 30-50 portraits overnight for events

### 中文
- Q 版风格 = FDM 友好几何（最小壁厚 >1.2mm，最大悬垂 <40°）
- 整体发块（非发丝）- $200 入门 FDM 可打印
- 2-5 分钟/个人像生成
- API 批量：过夜处理 30-50 个活动订单
