【使用方法 / How to Use】

1. 双击 "启动演示.bat" 或 "Launch.bat" 打开演示
   Double-click "启动演示.bat" or "Launch.bat" to start

2. 按 F11 进入全屏模式获得最佳体验
   Press F11 for fullscreen mode

3. 使用键盘或鼠标导航：
   Navigation:
   
   → 或 点击右半屏    下一页 / Next slide
   ← 或 点击左半屏    上一页 / Previous slide
   L                  切换语言 / Toggle language (中/EN)
   ESC                返回目录 / Back to directory

【文件结构 / File Structure】

├── 启动演示.bat      中文启动脚本
├── Launch.bat        English launcher
├── index.html        首页 / Home page
└── cases/            解决方案页面 / Solution pages

【系统要求 / Requirements】

- Windows 7/8/10/11
- 任意现代浏览器 (Chrome, Edge, Firefox)
- Any modern browser
